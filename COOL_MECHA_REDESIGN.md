## Technical Specifications

### File Details
- **Path**: `sprites/player_ship.svg`
- **Format**: SVG (Scalable Vector Graphics)
- **Dimensions**: 64×80 pixels
- **Viewbox**: 0 0 64 80
- **File Size**: 5,024 bytes (5KB)
- **Elements**: 100+ SVG shapes

### SVG Structure
- Uses paths for complex armor shapes
- Gradients via overlapping semi-transparent shapes
- Layered elements for depth
- Opacity effects for glowing elements
- Precise stroke widths for clean lines

## Customization Options

### To Change Colors
Edit `sprites/player_ship.svg`:

**Make it Blue:**
```svg
fill="#2C3E50"  →  fill="#2E5C8A" (Blue)
fill="#E74C3C"  →  fill="#3498DB" (Accent Blue)
```

**Make it Green:**
```svg
fill="#2C3E50"  →  fill="#27AE60" (Green)
fill="#E74C3C"  →  fill="#2ECC71" (Bright Green)
```
